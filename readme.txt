The plugin reacts to the active adblock plugin, prohibiting or limiting the user's capabilities.
== Description ==

###What does the plugin do?

The plugin detects ADBLOCK from the site visitor and offers several options for the development of events.
<ul>
<li>Hide specific text or block on the site</li>
<li>You can completely ban the viewing of the site</li>
<li>Politely ask to disable the adblock</li>
<li>A window that will not go to the site until the timer passes</li>
<li>Redirect visitor to another page</li>
</ul>  
To hide certain information on the site, it is necessary to highlight this and click on the adblock button in a text or visual editor
(A hand icon on a green background in a visual editor or an AOB button in a text editor)

== Installation ==

1. From your WordPress Dashboard go to 'Plugins' > 'Add new' and search for 'offadblock-wp'.
2. Click 'Install Now' under the title of the offadblock-wp plugin
3. Click activate
4. In the admin menu, find the plug-in icon offadblock-wp > configure the plug-in for your site > Save
5. Click 'Save' and you're done!

Or:

1. Upload `offadblock-wp`-folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the admin menu, find the plugin icon offadblock-wp > configure the plug-in for your site > Save
4. Click 'Save' and you're done!
